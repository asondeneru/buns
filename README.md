# Buns

Buns is good tamplate for building website with Remix React framework with yarn workspace.

## Usage

Install doppler see:
https://docs.doppler.com/docs/install-cli

```
$ cd apps/web
$ doppler login
$ doppler setup
$ doppler secrets download --no-file --format env > .env
```

Run web app:

```
$ yarn
$ yarn workspace web dev
```
